                          Hawthorn

  What is it?
  -----------

  Hawthorn provides a smart caching and lookup mechanism for bioinformatics-related
  ontology formats such as OBO, DAG and OWL.
  For examples of usage, please see test/src/uk/ac/ebi/hawthorn.

  Requirements
  ------------

  To install:
   o  A Java 1.4 or later compatible virtual machine for your operating system.
   o  Apache Ant 1.5.1 or later

  To run:
   o  A Java 1.4 or later compatible virtual machine for your operating system.
